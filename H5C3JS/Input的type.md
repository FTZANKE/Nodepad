### input的type：

| &lt;input type="这里都有啥？"&gt; | 描述                     |
| --------------------------------- | ------------------------ |
| text                              | 文本框                   |
| password                          | 密码输入框               |
| checkbox                          | 复选框                   |
| radio                             | 单选按钮                 |
| submit                            | 提交按钮                 |
| reset                             | 重置按钮                 |
| week                              | 日期按钮                 |
| time                              | 时间按钮                 |
| placeholder                       | 文本框里面的信息提示属性 |

- **去除焦点后的默认样式：**outline: none;
