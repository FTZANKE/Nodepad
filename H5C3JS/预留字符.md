### **预留字符：**

| 显示     | 描述     | 代码              | 编号      |
| -------- | -------- | ----------------- | --------- |
|          | 空格     | \&nbsp;           | `&#160;`  |
| <        | 小于号   | \&lt;             | `&#60;`   |
| >        | 大于号   | \&gt;             | `&#62;`   |
| &        | 和号     | \&amp;            | `&#38;`   |
| "        | 双引号   | \&quot;           | `&#34;`   |
| '        | 单引号   | \&apos;(IE不支持) | `&#39;`   |
| &cent;   | 分       | \&cent;           | `&#162;`  |
| &pound;  | 镑       | \&pound;          | `&#163;`  |
| &yen;    | 元       | \&yen;            | `&#165;`  |
| &euro;   | 欧元     | \&euro;           | `&#8364;` |
| &sect;   | 小节     | \&sect;           | `&#167;`  |
| &copy;   | 版权     | \&copy            | `&#169;`  |
| &reg;    | 注册商标 | \&reg;            | `&#174;`  |
| &trade;  | 商标     | \&trade;          | `&#8482;` |
| &times;  | 乘号     | \&times;          | `&#215;`  |
| &divide; | 除号     | \&divide;         | `&#247;`  |
